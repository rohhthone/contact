Diarium

Данная платформа предназначена для ведения личного блога или резюме в целях визуального отображения навыков. Тут была разработан специальный алгоритм для сравнения знаний относительно возраста. Данная платформа полностью написана по фронт энду и требует лишь дописания бэк энда чтобы стать полноценной рабочей моделью. При желании можно добавить наполнение организаций дабы показать сильнейшие стороны и конкурентные особенности. Данный проект может стать полноценной социальной сетью.
